# Wanderlust Explorer - Frontend Task

## Project Overview
**Wanderlust Explorer** is a travel-themed frontend project built with **HTML, CSS, and JavaScript**. The project features a responsive layout with an interactive image carousel, sliders, and a sidebar navigation menu, designed to showcase travel destinations and tips.

---

## Features
- **Responsive Layout**: Works smoothly on desktop, tablet, and mobile devices.
- **Sticky Header & Sidebar**: Fixed header with a collapsible sidebar menu for easy navigation.
- **Image Carousel**: Auto-sliding hero images for visual engagement.
- **Interactive Sliders**: Showcasing featured adventures and packing tips.
- **Buttons & Call-to-Action**: “Book a Trip” and “View Maps” buttons with hover effects.
- **Typography**: Clean fonts using Google Fonts (`Montserrat` and `Playfair Display`).
